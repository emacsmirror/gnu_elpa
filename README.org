* Installation
** Debian/Ubuntu
#+BEGIN_SRC bash
$ sudo apt-get install texlive make texi2html po4a texinfo
#+END_SRC
** MacOS
#+BEGIN_SRC bash
$ brew install texlive make texi2html po4a texinfo
#+END_SRC

* Create object files

#+BEGIN_SRC bash
$ make emacs-lisp-intro-es.html
$ make emacs-lisp-intro-es.pdf
$ make emacs-lisp-intro-es.info
#+END_SRC

* Clean files

#+BEGIN_SRC bash
$ make clean
#+END_SRC
